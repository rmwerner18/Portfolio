Projects

  • via GIPHY
    Chord Mixer is a project I completed for my capstone project at Flatiron. It allows users to select/customize chords, drums, and a melody, and hear it all played back to them.

  • via GIPHY
    SomeTrails is an app I created with a partner. It allows a user to find trails in their area, favorite them, and log hikes on them.

  • via GIPHY
    TweetCloud is another app I completed with a partner. It uses the Twitter API to render tweets by any twitter user. It then analyzes the tweets for most-used words, and displays these words in a word cloud, using size to indicate frequency. Users can then click on words, see their definitions and synonyms, and save the words to their work bank.
